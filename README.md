This started out as a simple repository to store dotfiles for three main programs I use - kitty, neovim, and btop. I would reinstall Nobara frequently, sometimes to get rid of bugs (thanks NVIDIA) and sometimes to give myself fresh start from all the bloat accumulated over a semester. Overtime, it has grown to accommodate a list of all the basic apps I need during re-install and other dotfiles. 

This will be a mostly stagnant, but everchanging repository. 

# Dotfiles
- **Terminal:** [Kitty](https://github.com/kovidgoyal/kitty)
- **Text Editor:** [Neovim](https://github.com/neovim/neovim)
- **Theme:** [Catppuccin](https://github.com/catppuccin/catppuccin)
- **Resource Monitor:** [Btop](https://github.com/aristocratos/btop)
- **GNOME extensions:** Blur My Shell, Just Perfection
